# pi4watering *** (In progress) ***
pi4j-project. (using java to controll the GPIO-ports on rPi)

Turns your raspberry-pi to a watering-system for flower-pots (indoor).

<ul>
<li>
  Raspberry pi
</li>
<li>
  Relay
</li>
<li>
